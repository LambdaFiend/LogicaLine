# Logica Line

### A secure Prolog based chat server built with your privacy in mind

# Features

  * E2E-Encryption built with RSA-2048 and SHA-128

  * Prolog Memory Safety

  * Minimalistic User Interface

# System Requirements

  * Version 1.0 runs on Linux

  * Install dependencies from src/requirements.txt

  * SWI-Prolog 9.0+

  * OpenSSL 1.1.1+
